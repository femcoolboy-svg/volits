# VOlits Messenger

Современный веб-мессенджер с реальным временем и неон/глассморфизм дизайном.

## Особенности

- 💬 Обмен сообщениями в реальном времени через WebSocket
- 🔐 Регистрация и аутентификация с капчей
- 👤 Профили пользователей и смена статуса
- 🤖 Встроенный бот-ассистент VOlits
- 🔍 Поиск пользователей
- 📱 Адаптивный дизайн для мобильных устройств
- 🎨 Стеклянный морфизм с неон-эффектами
